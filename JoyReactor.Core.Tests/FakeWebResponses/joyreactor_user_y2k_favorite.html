joyreactor android приложение 

JoyReactor Mobile

Рад представить вам еще одно приложение для просмотра постов мобильной версии Реактора с устройств на базе Android.


Основные возможности:

— Просмотр постов из основных разделов и тегов, комментариев к посту (включая скрытые)

— Голосование за пост/комментарий, добавление и ответ на комментарий

— Скачивание картинок/гифок из постов и комментариев

— Кэширование изображений, настройка максимального размера кэша

— Сохранение последних просмотренных постов, для просмотра в режиме оффлайн

— Копирование прямого URL на изображение/пост




Нажатие на карточку открывает просмотр комментариев, нажатие на картинку - ее просмотр.
Удерживайте на картинке, чтобы получить доступ к контекстному меню.

софт joyreactor android песочница в разработке 

Неофициальное приложение джоя для android

Привет, реакторчане!


В связи с тем, что на страницах вконтакте юмор уже, мягко говоря не торт, а в пути хочется почитать/посмотреть что-то интересное, решил сделать для себя приложение на андрюху ибо с телефона веб версию юзать трудновато (гифки, кубы, хайрез). В плей-маркете ничего на тему джоя не нашел. Вообще, планировалось для индивидуального использования (т.е. для меня лично), но я подумал, почему бы не сделать для всех?


Потому хочу провести опрос и выслушать комментарии. Авторизация пока не предусматривается, только read-only, так что секретные разделы пока недоступны (в виду того, что нет API).

Всё будет основано на парсинге и оптимальном кешировании ресурсов, чтобы при наличии хорошей мобильной связи (ещё лучше wifi) можно было юзать и радоваться.


Примерный функционал:

Просмотр

- Просмотр "Лучшего"

- Просмотр новых постов

- Просмотр разделов

- Просмотр определенного поста

- Просмотр случайного поста

- По выбору просмотр гифки/куба/ролика на ютубе

Сохранение

- Сделать закладку

- Сохранить гифку/изображение/пост

Другое

- Поиск

- Настройка обложки

- Настройка шрифта

- Управление/очистка ресурсов
